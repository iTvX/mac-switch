# Contributing

Mac Switch accepts changes through pull requests into `main`.

## Workflow

1. Create a feature branch from `main`.
2. Make a focused change.
3. Run `swift test` locally.
4. Open a pull request.
5. Wait for project review.
6. Merge after the change is accepted.

Official builds and release signing are handled by the maintainers. Do not
include local credentials, certificates, notarization output, build products, or
machine-specific release configuration in pull requests.
